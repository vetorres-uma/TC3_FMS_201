- Crear un proyecto PLC Standard en 01_Monolitico = Monolitico_PLC
- Cofigurar el proyecto
    - Habilitar Minimize ID changes
        - BD sobre el proyecto PLC > Properties > Common > Minimize Id changes in TwinCAT files
        * Evita los cambios de los número de identificación de los POU y sus objetos (metodos, acciones,...) incluidos en el archivo de cada POU. Lo  que evita que se produzcan cambios innecesarios de los que hay que hacer control de veriones.
    
    - Deshabilitar Write product version in files
        - BD sobre el proyecto PLC > Properties > Advanced > Write product version in files
        * Elimina la versión de TwinCAT (ProductVersion) con la que se abrío cada POU que parece que no aporta nada.

- Declarar todas las variables de entrada y salida en el bloque VAR de FB_Estacion
- Crear una visualización (V_Estacion)
